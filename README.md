# 🎮 Space Shooter — README

## Описание
**Space Shooter** — это простая 2D-игра, написанная на Python с использованием библиотеки Pygame. В игре вы управляете космическим кораблём (пандой), стреляете по врагам (апельсинам) и стараетесь не пропустить их мимо себя. Цель — набрать 10 очков или не допустить 3 промахов.

## Управление
- **← / →** — перемещение корабля влево и вправо
- **Пробел** — выстрел (ограничение по количеству выстрелов и перезарядка)

## Особенности
- Простая механика: движение, стрельба, подсчёт очков
- Враги появляются сверху и движутся вниз
- Перезарядка после 5 выстрелов
- Окончание игры при наборе 10 очков (победа) или 3 промахах (поражение)

## Требования
- Python 3.x
- Библиотека Pygame (`pip install pygame`)

## Запуск игры
1. Установите Python и Pygame.
2. Скачайте или склонируйте репозиторий.
3. Поместите в папку с проектом изображения:
   - `панда.png` (корабль)
   - `апельсин.png` (враг)
   - `колобок.png` (снаряд)
   - `galaxy.jpg` (фон)
4. Запустите файл игры:
   ```bash
   python main.py
   ```

## Файлы проекта
- **main.py** — основной код игры
- **панда.png** — изображение корабля
- **апельсин.png** — изображение врага
- **колобок.png** — изображение снаряда
- **galaxy.jpg** — фон игры
